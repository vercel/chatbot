# Research Capability

## Technique
1. **Search**: Use multiple query formulations across search engines
2. **Fetch**: Retrieve top N results, extract key content
3. **Cross-Reference**: Verify claims across 2+ independent sources
4. **Distill**: Produce structured summary with citations

## Anti-Patterns
- Never cite a single source for critical claims
- Never skip cross-referencing for compliance/legal topics

## Safeguards
- All claims must cite at least one source URL
- Mark confidence level (HIGH/MEDIUM/LOW) per claim
