# Affy Maverick Playbook

## Overview
Chargeback dispute management and defense automation via Maverick. Uses Base44 bridge for API access.

## Tools
- `getChargebacks` тАФ List chargebacks by status/customer
- `submitEvidence` тАФ Upload defense evidence
- `generateAffidavit` тАФ Auto-generate from transaction data
- `trackDispute` тАФ Track through resolution

## Auth
- `AFFY_API_KEY` via Base44 bridge

## Chargeback Workflow
1. Chargeback received тЖТ status: "open"
2. Evidence gathered тЖТ submitEvidence
3. Affidavit generated тЖТ generateAffidavit
4. Submitted тЖТ status: "under_review"
5. Outcome тЖТ status: "won" or "lost"

## Anti-Patterns
1. Never submit without evidence
2. Affidavits must include transaction receipts + IP logs
3. Track all deadlines тАФ chargebacks have strict response windows
