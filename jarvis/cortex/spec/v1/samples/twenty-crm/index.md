# Twenty CRM — Sample NKS Bundle

Demonstrates CRM as an OKF/NKS consumer:
- Twenty custom objects as OKF concepts
- Workflow mapping to Twenty automation
- RBAC integration with access levels
- Generative UI for admin panels
- Migration playbook generation from schema

**Files in this bundle:**
- `objects/customers.md` — Customer object schema
- `objects/deals.md` — Deal pipeline schema
- `objects/tasks.md` — Task management schema
- `workflows/sales-pipeline.md` — Sales workflow
- `permissions/rbac-mapping.md` — Role-to-access mapping
- `ui/admin-panel-schema.yaml` — Generative UI config

**NKS Features Demonstrated:** connector specs, RBAC, generative UI, workflow orchestration
