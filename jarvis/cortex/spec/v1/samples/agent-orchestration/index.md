# Agent Orchestration — Sample NKS Bundle

Demonstrates multi-agent coordination:
- Playbook router for intent-based agent dispatch
- Skill author for auto-generating new skills
- Mission dispatcher for task decomposition
- Agent orchestration playbook
- Session start/end handlers
- Intent routing workflow

**Files in this bundle:**
- `playbook-router.md` — Intent → agent routing
- `skill-author/SKILL.md` — Auto-skill generation
- `mission-dispatcher/SKILL.md` — Task decomposition
- `playbook-agent-orchestration.md` — Multi-agent coordination
- `workflows/intent-routing.yaml` — Intent routing pipeline

**NKS Features Demonstrated:** playbook routing, skill author templates, mission state machines, self-coding
