# GHL Patterns — Always-Check Rules

## Pre-Flight
- [ ] DNC check before any outreach
- [ ] Verify contact exists before creating opportunity
- [ ] Pipeline stage transition valid
- [ ] Contact has consented to communication type (SMS/email)
- [ ] Lead source tracked

## Pattern: Lead-to-Opportunity
1. Search/create contact
2. Qualify lead (score, source, intent)
3. Create opportunity in correct pipeline
4. Assign stage based on qualification
5. Track progression through stages
