# Hyperswitch Connector Skill

> **Connector:** hyperswitch | **Priority:** P1 | **Type:** Payment Gateway (Alternative)
> **Dependencies:** Hyperswitch API

## Purpose
Alternative payment gateway for high-risk BINs and payment link generation. Routes payments when NMI is not suitable (e.g., high-risk cards).

## When to Use
- High-risk BIN routing
- Payment link generation
- Alternative payment processing
- Refunds through alternate gateway
